Android 전용 운영체제입니다.
clear : 입력한 내용을 지웁니다.	info : 운영체제 정보를 표시합니다.
hwinfo : 하드웨어 정보를 표시합니다.	cd : 디렉터리를 이동합니다.
mkdir : 디렉터리를 만듭니다.	rmdir : 디렉터리를 삭제합니다.
shutdown : 운영체제를 종료합니다.	rename : 파일이름을 변경합니다.
rmfile : 파일을 삭제합니다.	filelist : 디렉터리에 파일을 리스트로 보여줍니다.
delcache : 캐쉬를 삭제합니다.	jpio : 'Just Print In Out' 의 약자로 편집이 아닌 그냥 출력'만' 합니다.

내장 프로그램
wpad : 문서를 편집할 수 있습니다.
calc : 정수만 계산할 수 있는 계산기 입니다.
