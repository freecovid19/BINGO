=====Codename BINGO=====
아직 OS의 이름이 정해지지 않았지만 'BINGO'라는 이름으로 시작해 임시로 썼습니다. BINGO는 BINGO Is Not GUI OS의 약자로 Wine의 Wine Is Not Emulator를 패러디해 지었습니다.

Android에서만 작동되게 만들었지만 소스코드를 바꾸어 Linux에서도 실행되게 할 수 있습니다. Windows 버전도 있습니다.

압축을 /storage/emulated/0에 풀어서 /storage/emulated/0/OS_Android가 되게 해주세요.

실행앱으로 Pydroid3를 추천드립니다. Pydroid3를 사용할때는 키보드를 Gboard로 사용하세요.
