000번대 오류코드 : 모듈오류. 100번대 오류와는 분리
100번대 오류코드 : 클라이언트 오류. 아래 방법대로 하고 안되면 보고바람.
200번대 오류코드 : 프로그램 상의 오류. 오류로그 파일을 개발자에게 보고해야함.

오류코드를 보는방법

'001C'라는 오류코드를 예시로 살펴보겠습니다.

오류코드는 숫자 3개와 알파벳 1개로 이루어져 있습니다. 숫자들은 오류의 고유 번호이고 알파벳은 오류의 심각성입니다. 

오류 고유번호

000번대 오류코드 :
001 : 필요한 모듈 없음
002 : 모듈 import 오류

100번대 오류코드 :
101 : BINGO가 예상된 위치에 존재하지 않음. 기본으로 설치된 경로에 파일을 옮기면 됨.

200번대 오류코드 :
201 : 로그파일을 그대로 개발자에게 전달해야함.

오류의 심각성
N : Nothing, 아무 위협없음
I : Ignore, 무시(심각하지 않음)
S  : Serious, 심각함
C : Critical, 치명적

위 표를 보고 '001C'를 해석하면 필요한 모듈이 없고 치명적 오류인걸 알 수 있습니다.
