000대 오류코드 : 모듈오류. 클라이언트 오류와 따로 분리함.
100대 오류코드 : 클라이언트의 오류. how_to_fix_error.txt로 자가진단후, 안된다면 개발자에게 도움요청 요함.
200대 오류코드 : 프로그램의 오류. 코드를 고쳐야 하므로 개발자에게 보고요함.

오류코드를 보는방법

'001C'라는 오류코드를 예시로 살펴보겠습니다.
오류코드는 숫자 3개와 알파벳 1개로 이루어져 있습니다. 숫자들은 오류의 고유 번호이고 알파벳은 오류의 심각성입니다. 

오류 고유번호

000대 오류코드 : 
001 : 필요한 모듈 없음
002 : 모듈 import중 에러

100대 오류코드 :
101 : BINGO가 예상된 위치에 설치되지 않았음.

200대 오류코드 :
201 : 프로그램 코드의 오류. 개발자에게 에러로그파일을 보고해야함.

오류의 심각성
N : Nothing, 아무 위협없음
I : Ignore, 무시(심각하지 않음)
S  : Serious, 심각함
C : Critical, 치명적

위 표를 보고 '001C'를 해석하면 필요한 모듈이 없고 치명적 오류인걸 알 수 있습니다.
